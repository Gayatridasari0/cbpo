This project is a simple Online Shopping system built using PHP, MySQL, HTML, and Admin Templates.
It includes database design, sample data, and a UI for performing basic CRUD operations.

---

Setup Instructions

Requirements:
- Install XAMPP (Apache + MySQL)
- Install any text editor (Notepad++, Sublime Text, or VS Code)
- Download and extract the project files

---

Steps to Run the Project:
1. Download and install XAMPP on your system.
2. Download the project ZIP file and extract it.
3. Copy the entire project folder (example: cbpo) to your XAMPP htdocs directory:
   C:/xampp/htdocs/
4. Open your web browser and go to:
   http://localhost/phpmyadmin
5. Create a new database named:
   cbpos_db
6. Import the SQL file provided:
   - Go to phpMyAdmin → Select your database → Click on Import tab → Choose the cbpos_db.sql file from the project files.
7. Now run the project in your browser:
   http://localhost/cbpo
